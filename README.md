# Sheria 18 - Portfolio

Bienvenue sur mon portfolio personnel.
Ce site présente mon profil, mes compétences, mes services, mes projets récents ainsi qu’un formulaire de contact fonctionnel.

## Aperçu du projet

**Sheria 18 - Portfolio** est un site web personnel développé avec **HTML, CSS et JavaScript**.
L’objectif est de présenter mon identité professionnelle, mes réalisations et mes compétences dans une interface moderne, responsive et personnalisable grâce à un système de thèmes.

## Lien du site

- Site en ligne : https://sheria-18-portfolio.onrender.com/
- Dépôt GitHub : https://github.com/ElSheria/Sheria-18---Portfolio

## Technologies utilisées

- HTML5
- CSS3
- JavaScript
- Font Awesome
- Formspree
- Render

## Fonctionnalités principales

- Page d’accueil moderne avec texte animé
- Section À propos avec présentation professionnelle
- Bouton **Voir mon CV** ouvrant le CV dans un nouvel onglet
- Section Services
- Section Compétences professionnelles
- Section Projets récents
- Section Témoignages
- Formulaire de contact fonctionnel avec Formspree
- Notification de message après envoi du formulaire
- Système de changement de couleur du thème
- Mode clair / sombre
- Bouton retour en haut de page
- Design responsive pour desktop, tablette et mobile
