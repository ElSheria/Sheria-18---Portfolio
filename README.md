# Sheria 18 - Portfolio

Bienvenue sur mon portfolio personnel.

Ce projet présente mon profil, mes compétences, mes services, mes projets récents ainsi que mes moyens de contact.
Le site a été développé avec **HTML**, **CSS** et **JavaScript**, avec une attention particulière portée au design, au responsive, aux animations et à l’expérience utilisateur.

## Aperçu du projet

**Sheria 18 - Portfolio** est un site web personnel conçu pour présenter mon identité professionnelle en tant que développeur web.

Le portfolio permet aux visiteurs de découvrir :

- mon profil professionnel ;
- mes services ;
- mes compétences techniques ;
- mes projets récents ;
- mon CV ;
- mes réseaux sociaux ;
- mes moyens de contact par formulaire et WhatsApp.

## Fonctionnalités principales

- Page d’accueil moderne avec texte animé
- Animation de chargement au lancement du site
- Navigation fixe avec menu responsive
- Section À propos avec présentation professionnelle
- Bouton **Voir mon CV** ouvrant le CV dans un nouvel onglet
- Section Services
- Section Compétences professionnelles avec filtres
- Section Projets récents
- Section Témoignages
- Formulaire de contact fonctionnel avec Formspree
- Notification après l’envoi du message
- Contact rapide via WhatsApp
- Système de changement de couleur du thème
- Mode clair / sombre
- Bouton retour en haut de page
- Animations d’apparition au scroll
- Design responsive pour ordinateur, tablette et mobile
